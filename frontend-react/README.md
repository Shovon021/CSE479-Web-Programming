# Flex & Bliss - React Frontend

Modern e-commerce storefront built with React + Vite.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

The Vite dev server proxies `/api` requests to the XAMPP backend.
See `vite.config.js` for proxy configuration.
